Usage (Windows): 
1. Run command prompt, preferably as administrator.
2. Change directory to path with "MyJSON_2" executable. 
3. Run it from command prompt, instruction format: 
	[PROGRAM] [INPUT_FILE_PATH] <OUTPUT_FILE_PATH> <option> <--help> <-h>
Example: MyJSON_2 --help
4. Run with --help key to see more information about usage.
